# 🤖 Intelligent Multi-LLM AI Assistant using LangChain

An AI-powered conversational assistant built using LangChain, Gemini API, and Groq LLMs.

This project integrates multiple Large Language Models (LLMs) with conversational memory and tool-calling capabilities to perform:
- Intelligent Question Answering
- Text Summarization
- Web Search
- Conversational AI Workflows

---

# 🚀 Features

✅ Multi-LLM Integration  
✅ Conversational AI Agent

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/PriyankaMittha/multi-llm-ai-assistant-langchain.git
```

---

## Navigate to Project Folder

```bash
cd multi-llm-ai-assistant-langchain
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Application

```bash
python app.py
```

OR for Streamlit:

```bash
python -m streamlit run app.py
```
